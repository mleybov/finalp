What is QA (quality assurance)?

Software testing is one of the most important steps before a product is released. There are so many things that can go wrong with the specifications or requirements that a programmar easily overlooks. Because of this, there are many jobs that are available in the tech field. There have been many deadly mistakes made because of software that was not tested properly.

In Mod 3, students placed in the QA track will learn about software testing and web development. The textbook used is called Software Testing ii Edition by Ron Patton. In these few weeks students learn the necessary tools to become a successful software tester in their internship.
